# Amazon Clone

A front-end clone of Amazon built while learning JavaScript through supersimpledev's Amazon course.

## About

This project was built as part of learning JavaScript fundamentals — DOM manipulation, working with arrays/objects, local storage, and structuring a multi-page site. Followed supersimpledev's tutorial and typed out the code myself to understand how everything works.

## Features

- Product listing page with search
- Add to cart / update quantity / delete from cart
- Cart total updates dynamically
- Checkout page with order summary
- Orders page showing order history
- Data persisted using localStorage

## Tech Used

- HTML
- CSS
- JavaScript (vanilla, no frameworks)
